# Nivel 1 -- Enunciado del Problema

Desarrolle un algoritmo en PSeInt que permita calcular el desempeño académico de **un estudiante**.

El algoritmo debe:

1.  Solicitar al usuario las tres notas del estudiante.
2.  Calcular el promedio de las tres notas.
3.  Determinar si el estudiante **APRUEBA** o **REPRUEBA**, teniendo en
    cuenta que:
    -   Si el promedio es mayor o igual a 3.0, el estudiante APRUEBA.
    -   Si el promedio es menor a 3.0, el estudiante REPRUEBA.
4.  Mostrar en pantalla el promedio obtenido y el estado final del estudiante.


# Nivel 2 -- Registro de Varios Estudiantes

Desarrolle un algoritmo en PSeInt que permita calcular el desempeño académico de varios estudiantes.

El algoritmo debe:

1.  Solicitar al usuario la cantidad de estudiantes que desea registrar.
2.  Para cada estudiante:
    -   Solicitar tres notas.
    -   Calcular el promedio.
    -   Determinar si el estudiante APRUEBA o REPRUEBA, teniendo en
        cuenta que:
        -   Si el promedio es mayor o igual a 3.0, APRUEBA.
        -   Si el promedio es menor a 3.0, REPRUEBA.
    -   Mostrar el promedio y el estado del estudiante.
3.  Al finalizar el registro de todos los estudiantes, mostrar:
    -   Total de estudiantes registrados.
    -   Total de estudiantes que aprobaron.
    -   Total de estudiantes que reprobaron.
    -   Promedio general del curso.
